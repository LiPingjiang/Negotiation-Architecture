# 附录C：谈判模拟练习与角色扮演脚本

> 从理论到实战的桥梁：通过模拟练习内化谈判技能

## 练习一：薪资谈判模拟

### 场景设定

**角色A：求职者（小李）**
- 5年经验软件工程师
- 目前年薪15万
- 期望年薪25万
- BATNA：留在原公司，或面试其他公司

**角色B：HR**
- 公司预算：18-22万
- 公司需求：急需技术人才
- BATNA：继续招聘，或降低要求

### 目标

- 角色A：争取最优薪资和福利
- 角色B：在预算内招聘到合适人才

### 时间

30分钟

### 准备（各10分钟）

**角色A准备：**
- 市场调研：同行业薪资水平
- 自我评估：技能、经验、价值
- BATNA分析：替代方案
- 策略设计：锚定、整合性谈判

**角色B准备：**
- 市场调研：人才市场供需
- 预算分析：薪资结构
- BATNA分析：替代候选人
- 策略设计：预算控制、价值展示

### 谈判流程

1. 开场（5分钟）：建立关系，设定议程
2. 报价（10分钟）：双方报价，锚定
3. 让步（10分钟）：讨价还价，交换
4. 收尾（5分钟）：达成协议或破裂

### 观察要点（旁观者）

- 谁先报价？锚定效果如何？
- 使用了哪些策略？
- 情绪管理如何？
- 是否达成双赢？

### 复盘问题

1. 双方的BATNA是什么？如何利用？
2. 锚定策略是否有效？如何改进？
3. 是否使用了整合性谈判？
4. 情绪管理如何？
5. 如果重新谈判，会如何改进？

## 练习二：商业合作谈判模拟

### 场景设定

**角色A：科技公司（甲方）**
- 有技术，需要渠道
- 希望独家合作
- 愿意分享50%利润

**角色B：渠道公司（乙方）**
- 有渠道，需要技术
- 希望非独家合作
- 希望70%利润

### 目标

- 角色A：达成独家合作，分享合理利润
- 角色B：保持灵活性，争取最大利润

### 时间

45分钟

### 准备（各15分钟）

**角色A准备：**
- 技术价值评估
- 渠道需求分析
- 竞争分析
- 合作方案设计

**角色B准备：**
- 渠道价值评估
- 技术需求分析
- 替代方案分析
- 合作方案设计

### 谈判流程

1. 开场（5分钟）：建立关系，设定议程
2. 利益探索（10分钟）：了解对方真正需求
3. 方案讨论（15分钟）：讨论合作模式
4. 讨价还价（10分钟）：利润分成、独家条款
5. 收尾（5分钟）：达成协议或破裂

### 观察要点

- 是否从立场转向利益？
- 创造性方案有哪些？
- 权力动态如何？
- 是否达成双赢？

### 复盘问题

1. 双方的真正利益是什么？
2. 创造性方案有哪些？
3. 权力动态如何影响谈判？
4. 如果引入第三方，会如何改变？

## 练习三：跨文化谈判模拟

### 场景设定

**角色A：美国公司代表**
- 直接沟通风格
- 短期导向
- 个人决策
- 详细合同偏好

**角色B：中国公司代表**
- 间接沟通风格
- 长期导向
- 集体决策
- 框架合同偏好

### 谈判议题

技术授权合作

### 目标

- 角色A：快速达成协议，详细条款
- 角色B：建立长期关系，框架合作

### 时间

45分钟

### 准备（各15分钟）

**角色A准备：**
- 了解中国文化特点
- 准备详细合同草案
- 设定明确目标
- 准备直接沟通策略

**角色B准备：**
- 了解美国文化特点
- 准备框架合作方案
- 设定长期目标
- 准备间接沟通策略

### 谈判流程

1. 开场（10分钟）：建立关系，文化适应
2. 议题讨论（15分钟）：技术授权范围
3. 条款讨论（15分钟）：合同细节 vs. 框架
4. 收尾（5分钟）：达成协议或破裂

### 观察要点

- 文化冲突如何表现？
- 双方如何适应对方文化？
- 沟通风格差异如何影响谈判？
- 是否达成双方满意的协议？

### 复盘问题

1. 文化差异如何影响谈判？
2. 双方如何调整沟通风格？
3. 直接 vs. 间接沟通的优劣？
4. 如何平衡效率与关系？

## 练习四：多方谈判模拟

### 场景设定

**三方：A公司、B公司、C公司**

共同投资项目，需要协商：
- 投资比例
- 利润分成
- 决策权
- 退出机制

### 角色设定

**角色A：**
- 资金最充裕
- 希望最大控制权
- 愿意承担最大风险

**角色B：**
- 技术最强
- 希望技术入股
- 希望保护知识产权

**角色C：**
- 渠道最广
- 希望渠道入股
- 希望快速变现

### 目标

- 三方达成协议
- 平衡各方利益
- 建立长期合作

### 时间

60分钟

### 准备（各20分钟）

**各角色准备：**
- 评估自身价值
- 分析其他两方
- 设计联盟策略
- 准备底线和BATNA

### 谈判流程

1. 开场（10分钟）：三方建立关系
2. 双边谈判（20分钟）：两两谈判
3. 三方谈判（20分钟）：共同讨论
4. 收尾（10分钟）：达成协议或破裂

### 观察要点

- 联盟如何形成？
- 权力动态如何变化？
- 议程如何控制？
- 是否达成三方满意？

### 复盘问题

1. 联盟策略如何影响结果？
2. 权力动态如何变化？
3. 议程控制的重要性？
4. 三方谈判与双方谈判的差异？

## 练习五：危机谈判模拟

- 公司面临财务危机

### 目标

- 角色A：在财务约束下达成协议

45分钟

- 财务数据分析

3. 利益探索（10分钟）：了解真正需求
4. 方案讨论（15分钟）：讨论解决方案

### 观察要点

- 僵局如何打破？
- 创造性方案有哪些？
- 是否达成双方满意？

### 复盘问题

1. 情绪管理如何影响谈判？
2. 僵局如何打破？
3. 创造性方案有哪些？
4. 危机谈判与常规谈判的差异？

## 练习六：家庭谈判模拟

### 场景设定

**角色A：丈夫**
- 希望购买新房
- 预算有限
- 希望靠近工作地点

**角色B：妻子**
- 希望购买新房
- 希望靠近孩子学校
- 希望有花园

### 目标

- 双方就购房达成一致
- 平衡工作、学校、预算

30分钟

### 准备（各10分钟）

**角色A准备：**
- 工作地点分析
- 预算分析
- 房源调研
- 优先级排序

**角色B准备：**
- 学校分析
- 花园需求分析
- 房源调研
- 优先级排序

### 谈判流程

1. 开场（5分钟）：表达需求
2. 方案讨论（15分钟）：讨论房源选择
3. 决策（10分钟）：达成一致

### 观察要点

- 利益如何平衡？
- 情绪如何管理？
- 创造性方案有哪些？
- 是否达成双赢？

4. 长期关系如何影响谈判策略？

## 练习组织指南

### 准备工作

1. **场地**：安静、舒适的会议室
2. **材料**：案例说明、角色卡、观察表
3. **时间**：预留充足时间（准备+谈判+复盘）
4. **人数**：2-6人（根据案例）

### 角色分配

- 明确角色和目标
- 保密信息（不透露给其他角色）
- 准备时间（10-20分钟）

### 观察与记录

- 指定观察者（不参与谈判）
- 记录关键行为、策略、情绪
- 使用观察表

### 复盘流程

1. **双方自评**：各自评价表现
2. **观察者反馈**：客观描述观察
3. **策略分析**：识别使用的策略
4. **改进建议**：如何改进
5. **经验总结**：学到的经验
