# 谈判架构

> *Negotiation Architecture: From Theory to Tactics*

---

## 引言

谈判无处不在——从薪资协商到商业并购，从家庭琐事到国际条约。大多数人把谈判当作口才的较量，但真正的谈判高手知道，谈判是一门**可以设计、可以架构、可以系统学习**的技艺。

本书将谈判视为一座建筑：

- **地基**——理论框架，理解谈判的底层逻辑
- **结构**——策略体系，掌握谈判的核心方法
- **装修**——话术技巧，打磨谈判的实战细节

从博弈论到心理学，从BATNA到锚定效应，从开场白到僵局处理，本书提供一套完整的谈判架构，让你在任何谈判桌上都能从容不迫、有的放矢。

## 写作风格

- **理论不空谈**：每个理论都配有具体案例和可操作建议
- **方法可落地**：每个策略都提供步骤清单和话术模板
- **案例跨场景**：涵盖商务、职场、消费、家庭、国际等多种情境
- **东西方融合**：既有西方博弈论的严谨，也有东方智慧的圆融
