# 第6章 谈判准备：信息收集、目标设定与BATNA构建

> "谈判的胜负，80%在谈判开始前就已经决定。" —— 迪帕克·马尔霍特拉

---

## 6.1 为什么准备如此重要

大多数人在谈判中的失败，不是因为技巧不够，而是因为准备不足。谈判准备不是"看看资料"，而是一个系统性的工程。

本章提供一个完整的谈判准备框架，帮助你在进入谈判室之前，就已经掌握了80%的胜算。

---

## 6.2 准备阶段一：信息收集

### 6.2.1 收集什么信息

谈判前的信息收集应该涵盖七个维度：

| 维度 | 具体内容 | 信息来源 |
|------|----------|----------|
| 对方利益 | 对方真正需要什么？他们的KPI是什么？ | 公开报告、行业分析、人脉网络 |
| 对方BATNA | 如果对方不跟我们谈，他们会做什么？ | 竞争对手动态、市场替代方案 |
| 对方约束 | 对方有什么时间、预算、政治约束？ | 新闻、财报、内部人士 |
| 市场标准 | 行业价格、条款、惯例是什么？ | 行业报告、公开交易、咨询专家 |
| 历史先例 | 类似谈判的结果是什么？ | 案例研究、公开记录、经验分享 |
| 对方风格 | 对方的谈判风格是合作型还是竞争型？ | 过往谈判经验、口碑、观察 |
| 决策流程 | 对方的决策需要谁批准？流程多长？ | 组织架构、LinkedIn、直接询问 |

### 6.2.2 信息收集的方法

**方法一：公开渠道**

- 公司官网、年报、新闻稿
- 行业报告、市场研究
- 专利、法律诉讼记录
- 社交媒体、LinkedIn

**方法二：人脉网络**

- 共同认识的人
- 行业协会、校友网络
- 前员工（注意伦理边界）
- 供应商、客户（如果合适）

**方法三：直接询问**

在谈判前的非正式交流中，可以直接询问：
- "您希望这次谈判达成什么目标？"
- "您的时间框架是怎样的？"
- "除了我们，您还在考虑哪些选择？"

注意：直接询问需要建立在信任基础上，不要像审讯一样提问。

**方法四：试探性提案**

通过提出一个试探性方案，观察对方的反应，从而推断他们的偏好和约束。

### 6.2.3 信息收集的伦理边界

信息收集是合法的，但以下行为可能越界：
- 商业间谍、窃取机密
- 虚假身份获取信息
- 利用内部人士的保密义务
- 黑客攻击、非法入侵

原则：你可以收集公开信息，可以询问合法问题，但不能使用非法或不道德的手段。

## 6.3 准备阶段二：目标设定

### 6.3.1 三层目标法

谈判目标不应该只有一个数字。好的谈判者设定三层目标：

```
        最优目标（Aspirational Target）
              |
        现实目标（Realistic Target）
              |
        保留点（Reservation Point）
              |
        谈判破裂
```

**最优目标（Aspirational Target）**
- 如果一切顺利，你能得到的最好结果
- 这个目标应该让你感到"有点不好意思"
- 它不是你的开价，而是你的"梦想结果"

**现实目标（Realistic Target）**
- 基于信息分析，你合理期望达成的结果
- 这是你谈判的核心目标
- 如果达成，你应该感到满意

**保留点（Reservation Point）**
- 低于此点，你宁可谈判破裂
- 保留点由BATNA决定
- 在谈判前明确，在谈判中坚守

### 6.3.2 目标设定的常见错误

**错误一：目标太低**

"我只要拿到平均价格就行"——这种心态让你从一开始就放弃了争取更好结果的可能。

研究表明，设定更高目标的人，即使最终没有达成最优目标，结果也往往比设定低目标的人更好。

**错误二：目标模糊**

"我希望得到一个好结果"——什么是"好"？没有量化，就没有方向。

**错误三：只有一个目标**

如果只有一个目标（如价格），谈判很容易陷入零和对抗。多目标谈判可以创造双赢空间。

### 6.3.3 多目标谈判的目标设定

在复杂谈判中，设定多个议题的目标：

| 议题 | 最优目标 | 现实目标 | 保留点 |
|------|----------|----------|--------|
| 价格 | 100万 | 120万 | 140万 |
| 付款方式 | 预付50% | 预付30% | 预付10% |
| 交货期 | 1个月 | 2个月 | 3个月 |
| 售后服务 | 3年免费 | 2年免费 | 1年免费 |
| 独家权 | 全国独家 | 区域独家 | 非独家 |

多目标谈判的优势：
- 可以在不同议题上交换让步
- 即使在一个议题上让步，可以在另一个议题上获得补偿
- 创造双赢空间

## 6.4 准备阶段三：BATNA构建

### 6.4.1 评估现有BATNA

在谈判前，诚实地评估你的BATNA：

**问题清单**：
1. 如果这次谈判破裂，我的最佳替代方案是什么？
2. 这个替代方案的价值是多少？
3. 实现这个替代方案的概率是多少？
4. 实现这个替代方案需要多长时间？
5. 这个替代方案的风险是什么？

### 6.4.2 改善BATNA

BATNA不是静态的。在谈判前，积极改善你的BATNA：

**策略一：寻找替代方案**

- 与其他潜在合作伙伴接触
- 即使你不打算用他们，他们的存在增强了你的议价能力
- 记住：你的BATNA不是"你打算做什么"，而是"你可以做什么"

**策略二：提升自身能力**

- 如果你可以自己解决问题，你就不依赖对方
- 投资内部能力建设，减少对外部供应商的依赖

**策略三：建立时间缓冲**

- 急迫感削弱BATNA
- 提前开始谈判，避免 deadline pressure
- 如果可能，设定多个备选时间表

**策略四：改善对方对你的感知**

- 让对方知道你有替代方案（但不要过度威胁）
- 展示你的市场价值
- 建立声誉，让其他潜在合作伙伴愿意与你谈判

### 6.4.3 评估对方的BATNA

谈判是双向的。了解对方的BATNA同样重要：

**问题清单**：
1. 如果对方不跟我们谈，他们的最佳替代方案是什么？
2. 这个替代方案的价值是多少？
3. 对方实现这个替代方案的难度如何？
4. 对方的急迫感如何？
5. 对方是否知道我们的BATNA？

**如何推断对方的BATNA**：
- 他们是否急于达成协议？（急迫感暗示弱BATNA）
- 他们是否愿意做出让步？（愿意让步暗示弱BATNA）
- 他们是否在寻找替代方案？（如果他们在找，说明他们的BATNA可能不强）
- 他们是否强调"独家"或"长期合作"？（可能暗示他们害怕失去你）

## 6.5 准备阶段四：策略设计

### 6.5.1 选择谈判风格

根据情境，选择你的谈判风格：

| 风格 | 特征 | 适用情境 |
|------|------|----------|
| 合作型 | 关注双赢，分享信息，创造选项 | 长期关系，整合性议题 |
| 竞争型 | 关注己方利益，隐藏信息，争取最大份额 | 一次性谈判，分配性议题 |
| 妥协型 | 双方各让一步，快速达成协议 | 时间紧迫，双方权力对等 |
| 回避型 | 推迟或退出谈判 | 议题不重要，或需要更多信息 |
| 适应型 | 优先考虑对方需求，维护关系 | 对方权力更大，关系更重要 |

注意：大多数谈判需要混合风格。根据议题的不同，在不同议题上采用不同风格。

### 6.5.2 设计开场策略

开场策略包括：

**谁先报价？**
- 先报价：设定锚点，控制谈判节奏
- 后报价：了解对方预期，避免锚定陷阱
- 选择取决于：你的信息优势、锚定策略、对方风格

**开什么价？**
- 极端开价：拉偏对方预期，但可能吓跑对方
- 温和开价：建立合作氛围，但可能放弃议价空间
- 现实开价：直接切入ZOPA，高效但可能缺乏策略性

**如何开场？**
- 建立关系：先聊非业务话题，建立信任
- 设定议程：明确谈判流程和议题
- 表达立场：清晰但不强硬地表达你的期望

### 6.5.3 设计让步策略

让步不是"让多少"，而是"如何让"。

**让步原则**：
1. **有条件**：每次让步都换取对方的让步
2. **递减**：让步幅度逐渐减小，暗示接近底线
3. **有理由**：为每次让步提供解释，避免显得软弱
4. **有节奏**：不要让步太快，让对方感到"来之不易"

**让步模式示例**：

```
第一轮：让10%（"考虑到我们的长期合作"）
第二轮：让5%（"这是基于您的订单量"）
第三轮：让2%（"这已经接近我们的成本底线"）
第四轮：让0%（"对不起，我们不能再让了"）
```

### 6.5.4 设计僵局策略

谈判中可能遇到僵局。提前准备破局策略：

- **暂停策略**："我们需要暂停一下，各自内部讨论"
- **引入新议题**："既然我们在价格上僵持，不如讨论一下付款方式？"
- **引入第三方**：调解人、仲裁者、专家意见
- **重新框架**："我们不是在争论价格，而是在讨论如何分配风险"
- **BATNA提醒**："如果无法达成协议，我们都将面临X后果"

### 6.5.5 设计收尾策略

如何锁定协议：

- **总结共识**："让我们确认一下已达成的共识..."
- **书面确认**：将口头协议写成备忘录或邮件
- **设定下一步**：明确后续行动和时间表
- **防范反悔**："为了确保双方理解一致，我们能否今天就签署意向书？"

## 6.6 准备清单

谈判前的最后检查清单：

### 信息准备
- [ ] 对方利益分析
- [ ] 对方BATNA评估
- [ ] 市场标准研究
- [ ] 历史案例分析
- [ ] 对方谈判风格了解

### 目标设定
- [ ] 最优目标明确
- [ ] 现实目标明确
- [ ] 保留点明确
- [ ] 多议题目标设定（如适用）

### BATNA准备
- [ ] 自身BATNA评估
- [ ] 自身BATNA改善行动
- [ ] 对方BATNA评估

### 策略设计
- [ ] 谈判风格选择
- [ ] 开场策略设计
- [ ] 让步策略设计
- [ ] 僵局策略设计
- [ ] 收尾策略设计

### 团队准备
- [ ] 谈判团队角色分配
- [ ] 内部沟通机制
- [ ] 授权范围明确
- [ ] 底线和 walk-away 信号

### 后勤准备
- [ ] 谈判地点和时间
- [ ] 资料和设备准备
- [ ] 备用方案（如果谈判延期或破裂）

## 6.7 本章小结

谈判准备是系统工程：

- **信息收集**：七个维度，四种方法
- **目标设定**：三层目标，多议题规划
- **BATNA构建**：评估、改善、推断对方
- **策略设计**：风格、开场、让步、僵局、收尾

记住：谈判的胜负，80%在谈判开始前就已经决定。准备充分的人，在谈判桌上从容不迫；准备不足的人，即使技巧再好，也难免被动。

下一章，我们将进入分配性谈判的策略世界——如何在零和博弈中争取最大份额。

## 练习与思考

1. 选择一个你即将面临的谈判，完成信息收集清单。
2. 设定三层目标：最优目标、现实目标、保留点。
3. 评估你的BATNA，并列出三个可以改善它的行动。
